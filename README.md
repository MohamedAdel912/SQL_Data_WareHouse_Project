# * Data Warehouse for ERP & CRM Systems

This project focuses on designing and building a **unified Data Warehouse** to centralize data coming from multiple operational systems, primarily **ERP** and **CRM** platforms. The goal is to provide a **clean, well-structured, analytics-ready environment** that supports accurate reporting, reliable dashboards, and scalable advanced analytics.

---

## * Project Objectives

* **Integrate Data Sources** from ERP and CRM into a single centralized warehouse.
* **Ensure Data Quality** through validation, cleaning, and transformation layers.
* **Enable Fast & Accurate Reporting** with optimized fact and dimension models.
* **Support Advanced Analytics** such as forecasting, segmentation, and performance tracking.
* **Reduce Operational Load** by offloading analytical queries from production systems.
* **Create a Single Source of Truth** for business users and analysts.
